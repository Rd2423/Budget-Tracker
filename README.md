
  # Budget Tracker 
  
  Budget Tracker is a simple web application that tracks your expenses and graphs monthly spending. It's built with vanilla JavaScript, HTML5 and CSS3.
  App uses service workers to allow effective offline experience.
  
  ## Link of the deployed app:
